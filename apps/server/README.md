# ⚙️ Apotza Server 🛠️


Working Node.js Server for Apotza Application with MongoDB, Express.js

## 📦 Dependencies

- [bcrypt](https://www.npmjs.com/package/bcrypt)
- [body-parser](https://www.npmjs.com/package/body-parser)

## 📦 Installation

- `npm i`

## 📦 Usage

- `npm run dev`
